Concepto clave

Versionar la API desde el inicio.

Problema

Cambios rompen clientes existentes.

Solución

Versioning.

Tipos

URI versioning
/api/v1/orders

Header versioning
X-API-VERSION

Media type versioning
application/vnd.company.v1+json

Buenas prácticas

Nunca eliminar versiones activas
Mantener backward compatibility
Documentar cambios
Deprecar versiones antiguas
Mantener roadmap de API

Recomendación

URI versioning es el más simple y usado.
